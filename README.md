# 🌟 Discover Mysore - Travel Website

A beautiful, modern travel website showcasing the top tourist attractions near Mysore, Karnataka, India. Built with HTML5, CSS3, and vanilla JavaScript with a responsive design that works on all devices.

## 🎯 Features

- ✨ **Modern Design** - Beautiful gradient backgrounds and smooth animations

## 📍 Attractions Included

1. **Chamundi Hills** - Sacred temple with panoramic city views
2. **Brindavan Gardens** - Magnificent terraced gardens with fountain light show
3. **Srirangapatna** - Historic island town with ancient temples
4. **Ranganathittu Bird Sanctuary** - Serene bird sanctuary on Kaveri River
5. **St. Philomena's Cathedral** - Stunning Neo-Gothic architecture
6. **Karanji Lake** - Picturesque lake for boating and sunsets
7. **Mysore Zoo** - One of India's oldest zoos with diverse wildlife
8. **Talakadu** - Ancient ruins and temples on Kaveri River banks

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No installation or dependencies required!

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/discover-mysore.git
   cd discover-mysore
   ```

2. **Open the website**

   **Option A: Python HTTP Server**
   ```bash
   python -m http.server 8000
   ```
   Then open: `http://localhost:8000`

   **Option B: Node.js HTTP Server**
   ```bash
   npx http-server
   ```

   **Option C: VS Code Live Server**
   - Install "Live Server" extension in VS Code
   - Right-click on `index.html` and select "Open with Live Server"

   **Option D: Direct File Opening**
   - Double-click `index.html` in your file explorer

## 🔧 Configuration

### Customize Attractions
Edit `js/script.js` to modify or add new attractions:
```javascript
{
  name: "Place Name",
  image: "images/image-name.jpg",
  description: "Description here",
  location: "Distance from city",
  rating: 4.7,
  type: "Category"
}
```

### Customize Colors
Edit `css/style.css` to change:
- Primary colors (gradients)
- Font families
- Animation speeds
- Card hover effects
